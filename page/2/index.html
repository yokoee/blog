10.1 节点层次

Node类型

JavaScript的所有节点类型都继承自 Node 类型。
每个节点类型都有一个 nodeType 属性,节点类型的定义在 Node 类型的常量中。

  • Document 类型
  • Element 类型
  • Text 类型
  • CDATASection 类型
  • DocumentType 类型
  • DocumentFragment 类型
  • Attr 类型

DOM 扩展

滚动

  • scrollIntoView(alignWithTop) 浏览器会滚动到自身顶部与元素顶部平齐。

  • scrollIntoViewIfNeeded(alignCenter) 当前元素在浏览器可视窗口不可见的情况下，浏览器会滚动到使元素可见。 alignCenter 为 true 时表示滚动到视口中间。

  • scrollByLines()

  • scrollByPages()

事件处理注意

发表于 2017-11-10 | 分类于 JavaScript
  • 事件处理函数中的 return ：
    • 当返回的是 true 时，将继续操作。
    • 当返回是 false 时，将中断操作。
    • 当不加return时将不会对 window.event.returnvalue 进行设置，所以会默认地继续执行操作。

3.4 引用

  • 对象通过引用来传递，它们永远不会被拷贝（包括Array）。
  • 对于不存在函数属性的对象，可以通过JSON的方法来实现深复制。

浅复制只复制一层对象的属性，而深复制则递归复制了所有层级。
如果对象存在函数属性，则该属性会被忽略，不会在新对象中出现。
