HTTP跨域请求

发表于 2017-04-18 | 分类于 HTTP

问题

在网页中向其它域名或端口请求资源时，就会发起跨域 HTTP 请求（cross-origin- HTTP request）。浏览器会限制脚本中发起的跨域请求（有些浏览器还不允许从 HTTPS 的域跨域访问 HTTP），但不会限制通过 HTML 标签中的 src 属性加载其它站点的资源。
